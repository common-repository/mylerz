== Description ==

###About mylerz

With the official mylerz Shipping app you profit from a quick and simplified shipment process enabling you to create pick up orders and generate AWBs. It covers domestic services from mylerz app supporting your e-commerce platform. Manage your shipments through one single shipping provider by using mylerz.

###Features:
Print AWBs with mylerz Woocommerce app: Fast and easy pick up creation for domestic orders as single or bulk creation process Create auto tracking numbers: Automatically attach tracking number to each order on Woocommerce orders page (if tracking is available for selected product) Reprint labels in case your printer experiences any issues Experience premium support, timely compatibility updates and bug fixes

This Shipping App offers currently mylerz major functions to facilitate:

= 1. Pick up creation. =

= 2. Print AWBs. =

= 3. Convert orders submitted to mark as fulfilled on Woocommerce. =

= 4. Link order fullfied by mylerz with its tracking number on Woocommerce orders page. =

= 5. Reprint previously fulfilled AWBs. =

= 6. Orders tags on mylerz app. =


###Prerequisites

You should have mylerz credentials, If you are a new customer, please contact us to create an account.

== Installation ==

1. Upload the plugin folder to the ‘/wp-content/plugins/’ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. That's it – you can now configure the plugin.
